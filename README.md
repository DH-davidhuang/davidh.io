# davidh.io

Personal website for David Huang — a single static page (`index.html`), no build step.
Served via GitHub Pages from the `main` branch.

To update: edit `index.html` directly and push. To add a publication, copy one `<li>` block
in the "Selected Publications" list.

The previous al-folio version is preserved on the `al-folio-backup` branch.
